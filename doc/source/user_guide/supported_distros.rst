Supported Distributions
=======================

Distributions which are supported as a build host:

- Centos 6, 7
- Debian 8 ("jessie")
- Fedora 20, 21, 22
- RHEL 6, 7
- Ubuntu 14.04 ("trusty")
- Gentoo


Distributions which are supported as a target for an image:

  - Centos 6, 7
  - Debian 8 ("jessie")
  - Fedora 20, 21, 22
  - RHEL 6, 7
  - Ubuntu 12.04 ("precise"), 14.04 ("trusty")
  - Gentoo
